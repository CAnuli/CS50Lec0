Git

This is used for:

  • Collaboration
  • Version control
  • Track changes
  • Synchronize code
  • COde testing while keeping original version

A repository is a central storage area for all the project files

How to create a repository on your local drive and also update github.

Steps:
  • Create repository inside GitHub
  • Copy HTTPS SSH url from inside GitHub: eg https://github.com/CAnuli/CS50Lec0.git
  • Go to terminal
  • Clone the repository onto your desktop ie locally: git clone ⟨ url ⟩ eg. git clone https://github.com/CAnuli/CS50Lec0.git
  • Change directory to the repository directory. eg, canuli$ cd CS50Lec0
  • Make your changes...eg. create index.html file CS50Lect0 canuli$ touch index.html
  • Add the file to the commit execution of the repository: git add ⟨ filename ⟩ eg. git add index.html
  • Take a snapshop of the repository: git commit - m "message" eg. git commit -m "added index.html"
  • Get the current status of the repository: git status
  • Send the changes to teh repository to Github: git push
  • Walaahh....Done!
